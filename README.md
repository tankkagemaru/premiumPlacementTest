# CEFR Placement Test

Adaptive English placement test for Premium Language Centre.

## Features

- Adaptive algorithm (Item Response Theory)
- 7 question types (multiple choice, listening, fill-in-blank, etc.)
- CEFR level placement (A1-C2)
- Email notifications
- Teacher dashboard with results & CSV export

## Quick Start

1. Sign up as teacher
2. Add questions to question bank
3. Students take test
4. Get CEFR placement
